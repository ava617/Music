    Quick way to enable/disable Javascript

    Support the next release on Kickstarter! (until January 24, 2019) https://www.kickstarter.com/projects/376707762/337761327 Quick Javascript Switcher is a Chrome extension that enable / disable JavaScript on the fly by hostname, and subdomain. (unlike to Chris Pederick's "Web Developer" extension who globally disables javascript) For example, disable javascript on www.google.fr not disable js on translate.google.fr Shortcut Windows [Alt+Shift+Q] Mac [Command+Shift+Q] The default shortcut can be customized in chrome://extensions/shortcuts Works with Chrome 16+ [FR] Quick Javascript Switcher est une extension permettant d'activer / dsactiver JavaScript la vole et par nom de domaine, et sous domaine.
